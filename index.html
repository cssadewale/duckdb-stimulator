⚙️ DuckDB Simulator Features

      
    
        
  • DuckDB-Style Execution Engine: Fast, in-browser analytical SQL engine optimized for heavy workloads, replicating DuckDB functionality.
    • 
        
  • Advanced Query Editor: Multi-tab environment with syntax highlighting, auto-completion, line numbers, and error-handling.
    • 
        
  • Visual Query Builder: Construct complex queries visually. Add SELECT columns and WHERE conditions without writing a single line of SQL.
    • 
        
  • CSV/Data Import & Export: Seamlessly load local datasets into your tables or export your query results as CSV for external analysis.
    • 
        
  • Macro Recording: Automate repetitive tasks. Record a sequence of commands and replay them anytime.
    • 
        
  • Data Visualization (Chart.js): Generate Bar, Line, Pie, and Doughnut charts dynamically from your query results without exporting data.
    • 
        
  • 🛡️ Enterprise Data Quality Profiler: Instantly generate data catalogs with completeness checks, null-ratios, uniqueness, and metric distribution without executing manual aggregation queries.
    • 
        
  • ⚡ Execution Plan Analyzer: Replicates DuckDB's 'EXPLAIN' capabilities. Visualizes query execution trees to help identify full-table scans, missing indexes, and performance bottlenecks.
    • 
        
  • 💻 DuckDB CLI Terminal (Exact Replica): For enterprise users accustomed to headless environments, we integrated a full Command Line Interface overlay. Use .help, .tables, .schema, and execute SQL directly via the console without touching the GUI.
    • 
        
  • 🧠 Live Memory Profiler: Integrated into the header, tracking real-time database RAM footprint (Wasm pages × page size). Essential for monitoring massive dataset loads locally.
    • 
        
  • Enterprise Security & Isolation: Zero-trust data environment. Datasets never leave the browser, ensuring strict compliance with internal data-handling policies.
    • 
        
  • Multi-Format Exports: Added native support for exporting query results directly to JSON and Markdown, alongside standard CSV.
    • 
        
  • Interactive Schema Explorer: Easily browse tables, views, and columns. Includes automatic row-count tracking and schema visualization.
    • 
        
  • Data Pagination & View: Handles thousands of rows gracefully with built-in pagination, column sorting, and copy-to-clipboard functionality.
    • 
        
  • Query History & Bookmarks: Never lose a query. Automatically saves execution history and allows you to bookmark frequently used scripts.
